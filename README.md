# pdf-translation-in-situ
对 PDF 进行基本保持排版的翻译，并支持原始页面与翻译页面之间的来回跳转

# 使用举例
python D:\My_Codes\PDF-translation-in-situ\main.py --en_cn_concat_only --use_openai
